## ⚠️ 需要你在 Supabase 执行一步操作

### 给 exams 表添加 module 字段

在 Supabase 控制台 → **SQL Editor** 中执行：

```sql
ALTER TABLE exams ADD COLUMN module TEXT NOT NULL DEFAULT 'listening';
```

这会让已有试卷默认归类到「听力」模块，新上传的试卷会按你选择的模块归入对应分类。

## 🆕 功能说明

### 1. 首页四模块切换
- 顶部四个 Tab：🎧 听力 | 📖 阅读 | ✏️ 写作 | 🎤 口语
- 每个 Tab 独立显示该模块的试卷
- 各模块有独立配色（蓝/绿/橙/紫）
- Hero 区域文案已改为"雅思全模块在线模考平台"

### 2. 管理员页面
- **上传时选择模块**：步骤1新增四模块按钮（必选）
- **管理试卷入口**：右上角「管理试卷」按钮
- **试卷列表页**：
  - 显示所有已上传试卷（标题、模块标签、时间、状态）
  - 每张卡片 hover 显示「预览」和「删除」按钮
  - 删除前有确认弹窗（防误删）
  - 删除同时清理关联的答题记录（attempts）
- **完成页新增**：「管理试卷」快捷入口

### 3. 数据库变更
- `exams` 表新增 `module` 列（TEXT，默认 'listening'）
- 支持4个值：listening / reading / writing / speaking
